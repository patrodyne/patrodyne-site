Patrodyne Site

This is a test page for developers. It provides two links:

  1. To check your Java installation,
  2. To launch the Hello installation.

The Hello installer is launched using Java Network Launching Protocol (JNLP). Before launching, the developer must edit patrodyne-site-hello-install.jnlp and set the codebase to the absolute path of the folder where patrodyne-site-hello-X.X.X-install.jar is stored.
